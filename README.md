# CodeAlpha_Chatbot

# 🤖 Basic Chatbot | Python

A Python-based command-line chatbot that responds to user inputs using predefined rules and interactive features.

## Overview

The **Basic Chatbot** is a rule-based Python application designed to simulate simple human conversations through predefined responses. Users can interact with the chatbot by entering various commands, and the chatbot responds with greetings, the current date and time, motivational quotes, jokes, Python tips, and interesting facts. This project demonstrates the practical implementation of Python fundamentals while providing an engaging and interactive user experience.

 ## Key Features

-  Personalized welcome message
-  Interactive text-based conversation
-  Displays current date and time
-  Random jokes
-  Motivational quotes
-  Python learning tips
-  Interesting facts
-  Built-in help menu
-  Exit command
-  Input validation for unsupported commands

## Getting Started

### Prerequisites

- Python 3.x installed on your computer

### Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Run the following command:

python chatbot.py

## Available Commands

| Command | Description |
|---------|-------------|
| `hello`, `hi`, `hey` | Greets the user |
| `how are you` | Checks the chatbot's status |
| `your name` | Displays chatbot information |
| `date` | Shows the current date |
| `time` | Shows the current time |
| `joke` | Displays a random programming joke |
| `quote` | Shows a motivational quote |
| `python` | Provides a Python learning tip |
| `fact` | Displays an interesting fact |
| `help` | Lists all available commands |
| `bye` | Ends the conversation |
